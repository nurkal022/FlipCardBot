# Vocabulary Cards Bot

Telegram-бот для изучения английской лексики с использованием ИИ для генерации карточек и SRS для повторения.

## Возможности

- Генерация карточек слов через OpenAI API
- Добавление слов в словарь с подтверждением
- Система повторения (SRS) с интервалами
- Два режима тестирования:
  - **Recall**: показать перевод → вспомнить слово
  - **Quiz**: выбрать правильный перевод из 4 вариантов
- Статистика изучения

## Установка

1. Клонируй репозиторий
2. Установи зависимости:
```bash
pip install -r requirements.txt
```

3. Создай файл `.env` на основе `.env.example`:
```bash
BOT_TOKEN=your_telegram_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
DATABASE_PATH=data/bot.db
```

4. Запусти бота:

**Обычный запуск (видно логи в консоли):**
```bash
cd /Users/nurlykhan/pets/FlipCardBot
python3 -m bot.main
```

**Запуск в фоне (работает в фоне, логи в bot.log):**
```bash
cd /Users/nurlykhan/pets/FlipCardBot
nohup python3 -m bot.main > bot.log 2>&1 &
```

**Остановка бота:**
```bash
pkill -f "bot.main"
```

**Проверка, что бот работает:**
```bash
ps aux | grep "bot.main" | grep -v grep
```

**Просмотр логов (если запущен в фоне):**
```bash
tail -f bot.log
```

## Использование

- Отправь боту слово или фразу на английском
- Бот сгенерирует карточку с переводом, определением и примерами
- Нажми "Добавить" для сохранения
- Используй `/review` для повторения слов
- Используй `/stats` для просмотра статистики
